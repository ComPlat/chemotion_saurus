:::info Supported file formats
For Mass spectrum analysis, we currently support [JCAMP-DX files](https://iupac.org/what-we-do/digital-standards/jcamp-dx/) (\*.jcamp, \*.dx, \*.jdx), \_.RAW (Thermo Fisher), \*.mz(X)ML, zip (in [BagIt](https://en.wikipedia.org/wiki/BagIt) format) files.
:::
