## 1. Supporting file formats

For Raman analysis, we are now only support [JCAMP-DX files](https://iupac.org/what-we-do/digital-standards/jcamp-dx/) (_.jcamp, _.dx, \*.jdx), zip (in [BagIt](https://en.wikipedia.org/wiki/BagIt) format) files.
