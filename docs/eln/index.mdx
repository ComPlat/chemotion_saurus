Hello and thank you for using Chemotion ELN!

If you are a first-time user, we recommend that you read the manual before you start working on your project, since it's not always possible to make the functionality of Chemotion ELN self-explanatory. In this manual, we will highlight what's necessary to make full use of the ELN's capabilities.

If you have specific questions, please check out the links below.
They correspond to the chapters of the manual:

- If you do not know where to start at all, go [here](eln/ui/first_steps).

- If you need support on how to report and structure your data, you can find videos [here](eln/videos).

- Please have a look at the [definition of terms](eln/ui) that we use throughout the manual.

- You can learn how to adjust your user interface [here](eln/ui/view).

- In the ELN, projects are organized in [collections](eln/ui/collections).

- Please also have a look at [using lists](eln/ui/lists).

- In the Chemotion ELN, you will work with elements,
  such as [samples](eln/ui/elements/samples), [reactions](eln/ui/elements/reactions) and [wellplates](eln/ui/elements/wellplates).

- [Detail modals](eln/ui/details_modal) provide information about your [structures](eln/ui/details_modal#details-modal-for-samples), [reactions](eln/ui/details_modal#details-modal-for-reactions) and [wellplates](eln/ui/details_modal#details-modal-for-wellplates).

- Learn about toolbar functions [here](eln/ui/toolbar).

- Questions about how to enter molecules into the Chemotion ELN will be answered in this [chapter](eln/ui/ketcher).

- With Chemotion ELN, you can [scan](eln/ui/barcodes#scan-barcodeqr-code) and [generate](eln/ui/barcodes#generate-barcodeqr-code) QR codes as well as bar codes.

- If you want to know how to create a group-inventory, follow this [link](eln/ui/inventory).

- The access of analytical data in the Chemotion ELN is done via ChemSpectra which is explained [here](services/chemspectra).

- Moreover, you can customize the ELN using [generic elements and segments](eln/admin/generic_config).

- [Here](eln/troubleshooting), you can find solutions for common issues.

While reading the manual, please mind the hints and instructions that are identified by the following symbols for <b>Caution</b> and <b>Recommendation</b>.
:::danger Caution
Very important facts you need to consider for a correct use of the Chemotion ELN.
:::

:::caution Recommendation
Important hints.
:::

:::info
Additional remarks and recommendations.
:::

:::note Links
Links that explain external tools and use of the Chemotion ELN.
:::

All terms in this manual that are written in <b> boldface </b> correspond to terms that are used in the user interface.

If your questions are not answered in the manual, please use the Helpdesk(/helpdesk) to contact us!
The ELN team is available for questions and assistance, but we do assume that you are familiar with this manual and,
in particular, with the highlighted <b>Cautions</b> and <b>Recommendations</b>.
