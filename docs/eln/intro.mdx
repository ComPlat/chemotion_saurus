Hello and thank you for using Chemotion ELN!

If you are a first-time user, we recommend that you read the manual before you start working on your project,
since it's not always possible to make the functionality of Chemotion ELN self-explanatory.
In this manual, we will highlight what's necessary to make full use of the ELN's capabilities.

If you have specific questions, please check out the links below.
They correspond to the chapters of the manual:

- If you do not know where to start at all, go [here](firststeps).

- Find information on the general background of the Chemotion ELN [here](about).

- If you need support on how to report and structure your data, you can find videos [here](videos_eln).

- If you need help accessing the Chemotion ELN, follow this [link](browser).

- Please have a look at the [definition of terms](structure) that we'll use throughout the manual.

- You can learn how to adjust your user interface [here](adjusting_views).

- In the ELN, projects are organized in [collections](collections).

- Please also have a look at [using lists](lists_eln).

- In the Chemotion ELN, you will work with elements,
such as [samples](samples), [reactions](reactions) and [wellplates](wellplates).

- [Detail modals](details_modal) provide information about your [structures](details_modal#details-modal-for-samples), [reactions](details_modal#details-modal-for-reactions) and [wellplates](details_modal#details-modal-for-wellplates).

- Learn about toolbar functions [here](toolbar_functions).

- Questions about how to enter molecules into the Chemotions ELN will be answered in this [chapter](ketcher).

- With Chemotion ELN, you can [scan](barcodes#scan-barcodeqr-code) and [generate](barcodes#generate-barcodeqr-code) QR codes as well as barcodes.

- If you want to know how to create a group-inventory, follow this [link](inventory).

- The access of analytical data via the Chemotion ELN is explained [here](analysis).

- Moreover, you can customize the ELN using [generic elements and segments](generic_config).

- [Here](problems), you can find solutions for common issues.


While reading the manual, please mind the hints and instructions that are identified by the following symbols for <b>Caution</b> and <b>Recommendation</b>.
:::danger Caution
Very important facts you need to consider for a correct use of the Chemotion ELN.
:::

:::caution Recommendation
Important hints.
:::

:::info
Additional remarks and recommendations.
:::

:::note Links
Links that explain external tools and use of the Chemotion ELN.
:::

All terms in this manual that are written in <b> boldface </b> correspond to terms that are used in the user interface.


If your questions are not answered in the manual, please [contact us](specific_contact)!
The ELN team is available for questions and assistance, but we do assume that you are familiar with this manual and,
in particular, with the highlighted <b>Cautions</b> and <b>Recommendations</b>.
