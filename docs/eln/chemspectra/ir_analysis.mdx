## 1. Supporting file formats
For NMR analysis, we are now only support [JCAMP-DX files](https://iupac.org/what-we-do/digital-standards/jcamp-dx/) (*.jcamp, *.dx, *.jdx), zip (in [BagIt](https://en.wikipedia.org/wiki/BagIt) format) files.
