The ELN follows the [semantic versioning](https://semver.org/) scheme. We strive to make releases on a regular basis. However, not all releases are packaged for external releases. After a round of feedback from internal users at KIT, only versions that are considered stable are released via the [chemCLI](install_configure/ChemCLI) tool.

## Latest Releases

Latest releases of the ELN, that are currently supported, are:
