The ELN follows the [semantic versioning](https://semver.org/) scheme. We strive to make releases on a regular basis. However, not all releases are packaged for external releases. After a round of feedback from internal users at KIT, only versions that are considered stable are released via the [chemCLI](install_configure/ChemCLI) tool.

## Latest Releases

Latest releases of the ELN, that are currently supported, are:

### Version 3

#### Version 3.0.0 (March 2026)

The new UI comes with several changes such as a new navigation structure, new UI adjustments (e.g. flexible change of the width), and changes to the reactions UI.

The new navigation structure includes a new panel below the summary of collections for collection management, and is drag and drop accessible via new symbols. Inbox, weighing tasks, calendar and notification are moved to collection management.

<YouTubeFrame
  src="https://www.youtube-nocookie.com/embed/aBNr3iJZdnM"

/><br/>

The reactions UI now provides additional information in the reaction table if more space is given to the reaction panel. The button for changing the reaction scheme information (samples given with name or structure) is moved to the reaction image, and my labels for reactions are now in scheme (not properties tab).

See the detailed release notes for version 3.0.0 [here](https://github.com/ComPlat/chemotion_ELN/releases/tag/v3.0.0).
