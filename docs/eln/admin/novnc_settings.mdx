# Access remote Desktops over VNC using websockify

installation: https://github.com/novnc/websockify

![adminui-vnc](@site/static/img/admin_ui/vnc.png)

Access to the remote can then be given to a user by associating it and the device within a user-group.
Administrators can assign devices and users to existing group (see [Groups & Devices](adui_group_devices)).

User can assign others to the groups they manage (see [My Groups & Devices](/docs/eln/inventory)).
