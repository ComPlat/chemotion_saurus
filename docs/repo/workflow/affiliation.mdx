During the review process, both submitters and reviewers can **modify the affiliation information associated with this submission**. They may add or remove affiliations for contributors and authors, as well as add or remove authors from the author list.
Additionally, they can assign more reviewers to this submission using the function introduced here.


## Authors & Reviewers button

You can access this function by navigating to: Review > Select a reviewing submission > Authors & Reviewers.
![](/img/repo/review/authors_and_reviewers_button.png)

## Add or Remove Contributor Affiliations

After clicking the Authors & Reviewers button, at the top of the interface, you can delete existing affiliations or add new affiliations for the **contributor**.
Please note that these changes apply **only** to this submission.
![](/img/repo/review/contributor_affiliation.png)

## Add or Remove Authors’ Affiliations

You can delete existing affiliations or add new affiliations for the **authors**.
Please note that these changes apply **only** to this submission.
![](/img/repo/review/authors_affiliation.png)

## Add Additional Reviewers

Before adding additional reviewers, you need to add them to your collaboration list.
You can do this using the **[My Collaborations](/docs/repo/settings_preparation#add-your-collaborators)** function.
Once added, select the desired collaborators and click the **Add to Reviewer List** button.
Please note that these changes apply **only** to this submission.
![](/img/repo/review/add-reviewers.png)

## Add Authors

Before adding additional authors, you need to add them to your collaboration list.
You can do this using the **[My Collaborations](/docs/repo/settings_preparation#add-your-collaborators)** function.
Once added, select the desired collaborators and click the **Add to Author List** button.
Please note that these changes apply **only** to this submission.
![](/img/repo/review/add-new-authors.png)

## Remove from Author List

You can remove an existing author from the author list by clicking the **Delete** button.
Please note that this change applies **only** to this submission.
![](/img/repo/review/remove_from_author_list.png)

## Remove from Reviewer List

You can remove an additional reviewer from the list by clicking the **Delete** button.
Please note that this change applies **only** to this submission.
![](/img/repo/review/remove_from_reviewer_list.png)

## Reload the ORCID iD

The ORCID iD is assigned when the submitter submits the submission, and it is retrieved from the collaborator’s account. If the ORCID iD is later updated in the collaborator’s account, you can click Get ORCID iD to reload the updated information.
Please note that this change applies **only** to this submission.
![](/img/repo/review/get_orcid_id.png)
