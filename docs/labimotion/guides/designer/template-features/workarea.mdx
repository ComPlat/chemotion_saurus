![Preview Design](/img/labimotion/designer-template-save.png)

<FigCaption text="Toolbar of Work area" />

### Upload template <FontAwesomeIcon icon={faUpload} />

You can upload a template file to the work area. It is a convenient way to create a template.

### Save and Release (Major) <FontAwesomeIcon icon={faFloppyDisk} />

You can save and release the template to the user page. The template will be in major release. For example, if the current version is `1.0`, the next version will be `2.0`.

### Save and Release (Minor) <FontAwesomeIcon icon={faFloppyDisk} />

You can save and release the template to the user page. The template will be in minor release. For example, if the current version is `1.0`, the next version will be `1.1`.

### Save as draft <FontAwesomeIcon icon={faFloppyDisk} />

You can save the template as a draft. The template will not be released to the user page.
