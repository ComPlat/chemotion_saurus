A **Text** type field is used for entering and displaying textual information. You can use this field to enter any short, written information.

<p align="center">
  <img src={fieldText} alt="LabIMotion - Text Field" />
</p>

:::tip
Text fields can be used in [Text Formula](text-formula) fields.
:::

## Extra Field Attributes

### Placeholder

A brief or example text that is displayed within the field.

### Read-only

The field is not editable.
