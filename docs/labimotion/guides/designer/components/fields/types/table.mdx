A **Table** type field presents structured data in a tabular format, allowing users to view, edit, and manage multiple records.

<p align="center">
  <img src={fieldTable} alt="LabIMotion - Table field" />
  <FigCaption text="Table field type" />
</p>

To set up an **Table** field, follow these steps:

1. Click the <FontAwesomeIcon icon={faSquarePlus} color="#337ab7" /> button to add the column.
2. Provide the heading in [Column Heading](#column-heading).
3. Specify its type from the dropdown menu in [Column Type](#column-type).
4. Give the initial value in [Default Value](#default-value).

In a **Table**, you can implement various field types including [Drag Molecule](drag-molecule), [Drag Sample](drag-sample), [Select](select), [System-Defined](system-defined), and [Text](text). For [Drag Sample](drag-sample), you have the options to display its name, external label, and molar mass as additional information. Similarly, for [Drag Molecule](drag-molecule), the available options include InChiKey, SMILES, IUPAC, and molar mass.

## Extra Field Attributes

### Id

A distinctive identifier within the table (generated by the system).

### Column Heading

The name of the column is presented to end-users.

### Column Type

The type of the column. The supported types are [Drag Molecule](drag-molecule), [Drag Sample](drag-sample), [Select](select), [System-Defined](system-defined), and [Text](text).

### Default Value

The initial value of the column. If the type of the column is [Drag Sample](drag-sample), you can specify to display the sample's name, external label, and molar mass as additional information. If the column type is [Drag Molecule](drag-molecule), the additional information could include InChiKey, SMILES, IUPAC, and molar mass.
