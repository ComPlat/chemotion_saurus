A **Text-Formula** type field provides a dynamic text output by combining multiple [Text](text) fields from this and other layers in one display. This field type is useful when you want to display a text that is a combination of multiple [Text](text) fields.The [Text](text) fields can be separated by different separators like space, comma, etc. The default separator is a dot (.).

<p align="center">
  <img src={textFormula} alt="LabIMotion - Text Formula" />
</p>

To set up a **Text-Formula** field, click the <FontAwesomeIcon icon={faSquarePlus} color="#337ab7" /> button to add the field that you want to combine and input the separator in the Separator field.

<p align="center">
  <img src={textFormulaProps} alt="LabIMotion - Text Formula" />

## Extra Field Attributes

### Id

A distinctive identifier within the table (generated by the system).

### Layer

The layer that contains the [Text](text) field that you want to combine.

### Field (type: text)

The [Text](text) field that you want to combine.

### Separator

The separator that separates the text fields. The default separator is a dot (.).
