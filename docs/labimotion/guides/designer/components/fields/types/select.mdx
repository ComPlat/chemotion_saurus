A **Select** type field offers users a dropdown menu with a curated list of predefined options. End-users can conveniently choose from this list, which consists of custom-defined values.

<p align="center">
  <img src={fieldSelect} alt="LabIMotion - Select Field" />
</p>

Create a dropdown list on the right side of the **[Work Area](../../../template-features/workarea.mdx)**. Enter the name of the select
list. Then, add the needed options. You can create multiple select lists and use them in
all layers of the element. It can be used as a restrictive field. Therefore, the values for
the restriction need to be entered as the names of the corresponding option(s).

<p align="center">
  <img src={fieldSelectOptions} alt="LabIMotion - Select Field" />
</p>
