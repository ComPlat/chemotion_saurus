The **Copy** function allows you to duplicate a selected element, creating an identical copy that retains the same attributes, settings, and content as the original. While utilizing this function, you will need to provide a value for the [Element/Name](./#elementname) field, which serves as the unique identifier for the newly copied element. This enables you to efficiently replicate complex content without starting from scratch.
