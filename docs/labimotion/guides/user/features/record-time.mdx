The **Record Time** function is available on each layer header. This feature allows users to manually record a specific date and time for reference.

The **Record Time** <FontAwesomeIcon icon={faClock} /> button is located at the top-right corner of the layer. When clicked, users can enter a desired date and time to associate with the layer. Once the data is saved, the record time will be displayed.

![RecordTime](/img/labimotion/record-time.png)

Simply click the <FontAwesomeIcon icon={faCircleXmark} /> icon to remove the record time.
