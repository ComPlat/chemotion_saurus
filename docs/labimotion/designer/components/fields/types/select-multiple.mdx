A **Select (Multiple)** type field allows users to select multiple options from a dropdown menu. This type field is useful when you want to allow users to choose several options from a list of predefined options.

  <img src={fieldSelectOptions} alt="LabIMotion - Select (Multiple) Field" />
  <FigCaption text="Select (Multiple) field type" />
</div>

## Set Up

**Select (Multiple)** is similar to the **Select** field. To set up a **Select (Multiple)** field, please refer to the [**Select > Set Up**](/docs/labimotion/designer/components/fields/types/select#set-up).

Note: The type should be set to **Select (Multiple).**
