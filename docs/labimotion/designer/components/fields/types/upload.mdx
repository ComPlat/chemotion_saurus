The **Upload** type field is used for uploading and storing files. A file can be uploaded via drag & drop or browsing the computer.

<img src={fieldUpload} alt="Generic Designer (menu)" />
<FigCaption text="Upload field type" />

For image files, an extended preview is available, allowing users to zoom in or out by adjusting the zoom percentage and to open the original image with a single click. For other file types, only the file name is displayed. All uploaded files are stored in the system and remain available for later reference or analysis.
