The **Drag Molecule** type features an icon with drag-and-drop functionality, allowing users to seamlessly add a molecule to a generic element by simply dragging and dropping it. Upon adding, the molecule image and its name is displayed for easy reference.

<div align="center">
  <img src={dragMolecule} alt="LabIMotion - Drag Molecule Field" />
</div>

:::info

The 'Drag Molecule' feature is only available for Generic Element and Segment. This limitation is due to the generic dataset being a modal view; hence, the drag function is not applicable in this context.

:::
