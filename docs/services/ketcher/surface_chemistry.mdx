# Solid Surface Chemistry

Ketcher Editor 3.6.x in Chemotion ELN supports **Surface Chemistry templates**. You can add pre-defined shapes to the canvas and attach text labels to them, which is useful for illustrating solid surfaces, supports, and related chemistry.

## Adding templates to the canvas

Surface Chemistry templates are added via the **Shapes** button in the Ketcher toolbar. A pre-defined list of templates is available.

![Shapes button in the toolbar](/img/ketcher/shapes_button.png)

### Template categories

Templates are grouped into two categories:

- **Basic templates** — General-purpose shapes for surfaces and supports.
- **Semantic templates** — Category-specific shapes with a defined chemical/semantic meaning.

Open the shapes list and choose a category to pick a template; click the desired template to add it to the canvas. You can place multiple shapes and move or resize them as needed.

## Adding labels to templates

After a shape is on the canvas, you can add a **label** (e.g. text or formula) using the **T** button next to the Shapes button.

![T button and label popup](/img/ketcher/t_button_popup.png)

### Steps to add a label

1. **Select a single shape** on the canvas (click the shape so it is selected).
2. Press the **T** button in the toolbar (next to the Shapes button).
3. In the popup window, enter the label text.
4. Click **Save**.

The label is attached to the selected shape. You can repeat these steps for other shapes. To change a label, select the shape again, press **T**, edit the text, and save.
