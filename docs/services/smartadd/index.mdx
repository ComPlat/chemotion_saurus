# SmartAdd

SmartAdd is a web-based companion for Chemotion that helps researchers import, organize, and prepare chemical data before sending it into Chemotion.
Instead of wrestling with complex file structures or scattered files, SmartAdd gives you a clear, visual tree view to curate your collection.

## Overview

SmartAdd streamlines the process of preparing research data for the Chemotion ELN. With its intuitive drag-and-drop interface and visual tree editor,
you can quickly organize samples, reactions, and analytical data before exporting to Chemotion.

## Quick Start

New to SmartAdd? Follow these steps:

### Manual Workflow:
1. **Upload Analytical Files**: Drag & drop analytical data files or ZIP archivesâ€”they appear in the input list (left)
2. **Create Structure**: Use "Add Sample" and "Add Reaction" buttons to build your collection manually (right panel)
3. **Assign Files**: Drag analytical files from the input list (left) to samples/analyses in the tree (right)
4. **Edit Properties**: Click nodes to edit sample/reaction properties, validate SMILES, assign roles
5. **Export**: Download as ZIP or upload directly to Chemotion ELN

### Auto-Generate Workflow:
1. **Upload Chemscanner ZIP or Excel Template**: Drag the file to auto-generate collection structure
2. **Review Structure**: Check auto-generated samples and reactions in the assignment tree (right)
3. **Assign Analytics**: Drag analytical files from input list to the generated samples/analyses
4. **Edit & Refine**: Adjust properties as needed
5. **Export**: Download as ZIP or upload directly to Chemotion ELN

### ğŸŒ³ Right Workspace - Tree View & Editor (Data Preparation)
- **Action Buttons**: Add Sample, Add Reaction and Export Collection
- **Tree View**: Hierarchical display of collections (samples, reactions, analyses)
- **Clickable Nodes**: Select items to view/edit properties
- **Properties Editor**: Edit selected item details
- Expand/collapse nodes to explore hierarchy
- Select a node to edit its properties
- Changes save instantly (No need to update button)

## Supported Data Types

SmartAdd works with various chemical data formats:

### Analytical Data Files (Input List)
- **Spectra**: NMR, IR, data files
- **Images**: PNG, JPG, TIFF
- **Raw Data**: CSV, TXT, vendor-specific formats
- **Archives**: ZIP files containing analytical data

### Structure Import Files (Auto-Generate Collections)
- **Chemscanner Exports**: ZIP files with export.json (auto-generates samples/reactions)
- **Excel Templates**: .xlsx files following our template format with structures and metadata (auto-generates collections samples/reactions)
- **MOL Files**: Molecular structure files
