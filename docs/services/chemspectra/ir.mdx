:::info[Supported file formats]

For Infrared analysis, we currently support [JCAMP-DX files](https://iupac.org/what-we-do/digital-standards/jcamp-dx/) (\*.jcamp, \*.dx, \*.jdx), zip (in [BagIt](https://en.wikipedia.org/wiki/BagIt) format) files.

:::
