title: Dynamic Light Scattering (DLS)

:::info Supported file formats
For Dynamic Light Scattering analysis, we currently support [JCAMP-DX files](https://iupac.org/what-we-do/digital-standards/jcamp-dx/) (\*.jcamp, \*.dx, \*.jdx), zip (in [BagIt](https://en.wikipedia.org/wiki/BagIt) format) files.
:::

To upload and view your data, you can follow [this instruction](../chemspectra#using-chemspectra).
