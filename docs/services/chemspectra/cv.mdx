:::info Supported file formats
For Cyclic Voltammetry analysis, we currently support [JCAMP-DX files](https://iupac.org/what-we-do/digital-standards/jcamp-dx/) (\*.jcamp, \*.dx, \*.jdx), zip (in [BagIt](https://en.wikipedia.org/wiki/BagIt) format) files.
:::

1. To upload and view your data, you can follow [this instruction](../chemspectra#using-chemspectra).

2. Select which spectrum that you want to view:
   ![chemspectra cv select curve](/img/chemspectra/cv/chemspectra_cv_1.png)

3. Select which spectrum that you want to work with:
   ![chemspectra cv select curve to work](/img/chemspectra/cv/chemspectra_cv_2.png)

4. Select to add max and min peaks:
   ![chemspectra cv add peaks](/img/chemspectra/cv/chemspectra_cv_3.png)

5. Select to add peckers:
   ![chemspectra cv add peckers](/img/chemspectra/cv/chemspectra_cv_4.png)

6. **Optional** Download your analyzed data as table format:
   - Open analyses tab, scroll down to `Datasets` section, click on which data you want to see the details:
     ![chemspectra cv view open dataset](/img/chemspectra/cv/chemspectra_cv_5.png)
   - Select your analyzed `csv` file to download:
     ![chemspectra cv select csv](/img/chemspectra/cv/chemspectra_cv_6.png)
     ![chemspectra cv csv](/img/chemspectra/cv/chemspectra_cv_7.png)
