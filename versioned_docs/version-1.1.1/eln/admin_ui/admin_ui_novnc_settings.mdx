# Access remote Desktops over VNC using websockify

installation: https://github.com/novnc/websockify

![adminui-vnc](/img/adminui-vnc.png)

Access to the remote can then be given to a user by associating it and the device within a user-group.
Administrators can assign devices and users to exisiting group (see [Groups & Devices section](/docs/eln/admin_ui/groups_n_devices)).

User can assign others to the groups they manage (see [My Groups & Devices](/docs/eln/)).
